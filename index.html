Scrap

Scrap is a new block-based programming language aimed at advanced users. It is written in pure C and is inspired by other block-based languages such as Scratch and its forks like Turbowarp.

Notable Advantages from Scratch

Faster runtime (still not faster than Turbowarp as Scrap is interpreted for now)

The code runs in a separate thread, solving some performance issues compared to Scratch.

New, more advanced else if, else blocks, reducing nested checks in if-else blocks (a more flexible variant of Snap's Snap! if-else blocks)

Variables can have a lifetime, avoiding name conflicts and allowing temporary variables.

Custom blocks can return values and be used as arguments for other blocks.

Various string manipulation and bitwise operator blocks.

Data type conversion functions.

More strict equality checks for [[ ]] = [[ ]] and [[ ]] != [[ ]] blocks (case-sensitive and data-type aware).

Lists are now a data type instead of a different variable type, allowing nested lists (though currently inconvenient).
